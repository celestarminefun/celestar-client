## Features
* Auto Fullscreen
* Auto Updates
* Badges
* Built-in Ad Blocker
* Custom Crosshair
* Custom Mod Menu
* Discord Rich Presence
* Join Link Button
* `sandbox.minefun.io` Support
* Screenshot
* Texture Pack
* Zoom
### Planned

## Keybinds

| Key | Action |
|-----|--------|
| `G` | Open Client Menu |
| `F2` | Screenshot to clipboard |
| `F4` | Return to `https://minefun.io` |
| `F5` | Reload |
| `F6` | Load URL |
| `F7` | Copy URL |
| `F11` | Toggle Fullscreen |
| `F12` | Open DevTools |
