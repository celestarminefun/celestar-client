## Features
* Auto Updates
* Built-in Ad Blocker
* Custom Crosshair
* Custom Mod Menu
* Discord Rich Presence
* Join Link Button
* Skip Loading Screen
* `sandbox.minefun.io` Support
* Zoom
### Planned
* Colourful Chests
* Hunger Value
* Import/Export Client Settings
* Keystrokes
* QOL Features

## Keybinds

| Key | Action |
|-----|--------|
| `G` | Open Client Menu |
| `F4` | Return to `https://minefun.io` |
| `F5` | Reload |
| `F6` | Load URL |
| `F7` | Copy URL |
| `F11` | Toggle Fullscreen |
| `F12` | Open DevTools |
